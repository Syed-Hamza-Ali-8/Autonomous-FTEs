## Section Guidelines

### Executive Summary
- 2-3 sentences maximum
- Focus on most important highlights
- Include 4 key metrics with trends
- List 1-2 critical action items

### Financial Summary
- Revenue, expenses, profit, margin
- Outstanding invoices table
- Cash flow trend (3 weeks minimum)
- Brief insight on financial health

### Social Media Performance
- Engagement metrics table (all platforms)
- Top 3 performing posts
- 3 key insights about social performance

### Communications Activity
- Email, WhatsApp, LinkedIn metrics
- Response times vs targets
- Pending items count

### Task Performance
- Completion rate vs target
- Bottlenecks with impact assessment
- Time tracking breakdown

### Subscription Audit
- Active subscriptions with usage
- Unused services flagged
- Potential savings calculated

### Insights & Recommendations
- 4-6 insights categorized by type
- Each with issue, impact, action, priority
- Use emojis for visual categorization

### Action Items
- Categorized by priority (High/Medium/Low)
- Checkbox format for tracking
- Specific and actionable

### Upcoming Deadlines
- Next 2-4 weeks
- Critical dates only

- `YYYY-MM-DD`: Date in ISO format
- `[amount]`: Dollar amount

- `[#]`: Number/count
